# Proyecto-Analisis_de_un_Lenguaje_de_Programacion
Proyecto- Análisis de un Lenguaje de Programación en Php. Aplicación que calcula promedio de notas de los estudiantes


Intrucciones Compilación:
   1) Poner todos los archivos en un directorio (Incluido el csv)
   2) Tener la direccion del directorio en la terminal del directorio
   3) Ejecutar:
      ```bash
       php main.php
      ```
